# פלטפורמת לימוד מדעים — כיתה ט'

כלי עזר אישי להוראה, שבניתי כדי ללוות את תלמידי כיתה ט' שלי בשיעורי מדעים. הפלטפורמה
משלבת תוכן לימודי עם מורים דיגיטליים מבוססי בינה מלאכותית (Gemini Gems), בהמשך לגישה
שפיתחתי בפרויקט "החשמלאי הראשי".

## על הפרויקט

תלמידים לומדים מדעים בצורה מיטבית כשיש להם מרחב לתרגל, לטעות, ולקבל משוב מיידי ומותאם
אישית — לא רק הרצאה חד-כיוונית. הפלטפורמה נבנתה כדי לתת לכל נושא לימודי "תחנת עבודה"
משלו: הסבר קצר, כלי אינטראקטיבי לתרגול, ומורה דיגיטלי סוקרטי שמלווה את התלמיד בשאלות
מנחות במקום לתת לו תשובות ישירות.

## מבנה הריפו

- `index.html` — עמוד הבית של הפלטפורמה, עם כרטיסי תחנות לכל נושא/מורה
- (יתווספו בהמשך) עמודי נושא נפרדים וכלים אינטראקטיביים לכל תחנה

## הרצה
